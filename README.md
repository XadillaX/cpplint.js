# CPPLINT.js

Node.js wrapper for [CPPLINT](https://github.com/cpplint/cpplint).

## From the command-line

```shell
$ npm install -g cpplint.js
$ cpplint --help
```

You can go https://github.com/cpplint/cpplint to get more information.
